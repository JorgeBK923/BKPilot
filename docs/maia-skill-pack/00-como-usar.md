# Como usar o MAIA Skill Pack na prática

## Regra de ouro
Você não conversa com a IA de forma solta. Você chama a skill correta para a fase correta.

## Exemplo de fluxo
1. MAIA Diagnóstico - entender o projeto.
2. MAIA Especificação - transformar ideia em requisitos.
3. MAIA Planejamento - quebrar em tarefas.
4. MAIA Arquitetura - decidir estrutura técnica.
5. MAIA Harness - preparar ambiente para IA.
6. MAIA Implementação - executar uma tarefa.
7. MAIA QA Validação - testar e validar.
8. MAIA Review - revisar riscos e qualidade.
9. MAIA Release - preparar entrega.
10. MAIA Memória - registrar aprendizado.

## Comando base
```text
Use a skill [NOME DA SKILL] para [OBJETIVO]. Siga os limites da skill, gere os artefatos obrigatórios e atualize progresso.md e decisions.md quando necessário.
```
