# MAIA Skill Pack v1

O MAIA Skill Pack transforma o método MAIA em um conjunto de skills práticas para coordenar CLIs e agentes de IA, como Codex, Claude Code, OpenCode e ferramentas semelhantes.

## Objetivo
Padronizar como a IA entende, planeja, implementa, valida, revisa e registra decisões em projetos de software.

## Uso rápido
1. Copie esta pasta para o projeto.
2. Abra a CLI de IA desejada.
3. Chame a skill adequada.
4. Entregue o contexto da demanda.
5. Revise o resultado, os arquivos alterados e os checklists.

Exemplo:
```text
Use a skill MAIA Especificação para transformar esta ideia em requisitos, regras de negócio e critérios de aceite.
```
