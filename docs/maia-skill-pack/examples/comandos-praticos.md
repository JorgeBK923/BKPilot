# Comandos práticos para usar nas CLIs

## Diagnóstico
```text
Use a skill MAIA Diagnóstico para analisar este repositório e gerar diagnostico.md, riscos-iniciais.md e perguntas-pendentes.md. Não implemente nada.
```

## Especificação
```text
Use a skill MAIA Especificação para transformar a ideia abaixo em requisitos, regras de negócio e critérios de aceite.
```

## Implementação
```text
Use a skill MAIA Implementação para executar apenas a etapa 1 do plano-execucao.md. Altere somente o necessário e atualize progresso.md.
```

## QA
```text
Use a skill MAIA QA Validação para gerar cenários manuais, checklist de regressão e riscos de qualidade da mudança feita.
```

## Review
```text
Use a skill MAIA Review para revisar os diffs, riscos, aderência ao escopo e documentação antes do merge.
```
